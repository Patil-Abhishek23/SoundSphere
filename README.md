# SoundSphere Music Website

## Description
SoundSphere is a music website developed using Java Spring framework, incorporating Razorpay payment integration for users to access premium features. The project utilizes Thymeleaf for front-end development, employing HTML, CSS, and MySQL for database management. SoundSphere caters to two types of users: regular users and administrators.

## Features
- **User Authentication**: Users can register, login, and manage their accounts securely.
- **Premium Access**: Users can upgrade to premium membership using Razorpay payment integration to access exclusive features.
- **Admin Panel**: Administrators have special privileges users, content, and other aspects of the website, including adding new songs and playlists.
- **Music Catalog**: Users can explore a diverse catalog of music tracks, albums, and artists.
- **Playlist Creation**: Users can create and manage personalized playlists.
- **Responsive Design**: The website is designed to be accessible and functional across various devices and screen sizes.

## Technologies Used
- Java Spring framework
- Thymeleaf
- HTML
- CSS
- Hibernate

## Setup Instructions
1. Clone the repository to your local machine.
2. Ensure you have Java Development Kit (JDK), Apache Maven, and MySQL installed.
3. Import the project into your preferred IDE.
4. Configure the MySQL database settings in the application.properties file.
5. Run the application using Maven or your IDE's built-in tools.
6. Access the website through the provided localhost URL.
7. Explore the features and functionalities of SoundSphere!
